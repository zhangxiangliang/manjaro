# Drivers - 驱动

## Realtek 8852AE 网卡驱动

```
git clone git://github.com/lwfinger/rtw89.git
cd rtw89
make
sudo make install
```

> 注意: 编译需要安装相关依赖

> 注意: 相关依赖和内核版本是否一致

> 注意: 可以使用拓展钨或者手机 usb 接入提供网络临时救急

> 注意: 网络/蓝牙/不爱折腾 好可以换一个网卡 [AX210-M](https://u.jd.com/UthaHot)

> 关键词：小米笔记本电脑/红米笔记本电脑/螃蟹卡/网卡识别不到 （这里只是方便大家搜索得到）
